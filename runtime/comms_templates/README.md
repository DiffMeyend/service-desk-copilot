# Comms Templates

Context-aware communication templates for ticket notes, customer emails,
and escalation handoffs.

## Sources
- Export your Text Blaze snippets as JSON and drop them here as `textblaze_export.json`
- Hand-authored templates live in `templates.yaml`

## Usage
When working a ticket, ask QF_Wiz to draft a note or email using a template.
The GPT will pull from this directory and fill in context from the active ticket.
